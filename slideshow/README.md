# The JavaScript plugin - slideshow

## Using
1. Add the data tag to the block what you want to setup the slider:
data-slideshow="true"

2. Example structure:
< ul data-slideshow="true" >
	< li >< img src="..." / >< /li >
	....
< /ul >

## Functionalities
- init()
- setup()
- play()
- pause()
- next()
- prev()
- destroy()
- onAfterNext()
- onAfterPrev()
